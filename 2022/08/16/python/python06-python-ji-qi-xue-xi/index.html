一、线性回归

1.1 回归问题概述

例：

  • 数据：工资和年龄（2个特征）
  • 目标：预测银行会贷款给我多少钱（标签）
  • 考虑：工资和年龄都会影响最终银行贷款的结果，那么它们各自有多大的影响？（参数）

通俗解释：

  • X1,X2就是我们的两个特征（年龄，工资)

  • Y是银行最终会借给我们多少钱

  • 找到最合适的一条线（想象一个高维)来最好的拟合我们的数据点

假设$\theta_{1}$是年龄的参数，$\theta_{2}$是工资的参数，拟合的平面是：$h_{\theta}(x) = \theta_{0}+\theta_{1}x_{1}+\theta_{2}x_{2}$($\theta_{0}$为偏置项)，整合为：

$h_{\theta}(x) = \sum_{i=0}^{n}\theta_{i}x_{i} = \theta^{T}x$(其中$x_{0} = [1,…,1]$)

对于每个样本，真实值和预测值存在差异，即有：$y^{(i)} = \theta^{T}x^{(i)}+\varepsilon^{(i)}$

1.2 独立同分布的含义

误差$\varepsilon^{(i)}$是独立并具有相同的分布，且服从均值为0、方差为$\theta^{2}$的高斯分布。

高斯分布：银行可能会多给，也可能会少给，但绝大多数情况下这个浮动不会太大，极小情况下浮动会比较大，符合正常情况。

1.3 似然函数

预测值与误差：$y^{(i)} = \theta^{T}x^{(i)}+\varepsilon^{(i)}$

由于误差服从高斯分布：$p\left(\epsilon^{(i)}\right)=\frac{1}{\sqrt{2 \pi} \sigma} \exp \left(-\frac{\left(\epsilon^{(i)}\right)^{2}}{2 \sigma^{2}}\right)$

将(1)式代入(2)式：$p\left(y^{(i)} \mid x^{(i)} ; \theta\right)=\frac{1}{\sqrt{2 \pi} \sigma} \exp \left(-\frac{\left(y^{(i)}-\theta^{T} x^{(i)}\right)^{2}}{2 \sigma^{2}}\right)$

注：$\theta$和$x^{(i)}$进行组合后，与$y^{(i)}$（真实值）越接近越好。

似然函数：$L(\theta)=\prod_{i=1}^{m} p\left(y^{(i)} \mid x^{(i)} ; \theta\right)=\prod_{i=1}^{m} \frac{1}{\sqrt{2 \pi} \sigma} \exp \left(-\frac{\left(y^{(i)}-\theta^{T} x^{(i)}\right)^{2}}{2 \sigma^{2}}\right)$

注：

  • 独立同分布的前提下，联合概率密度等于边缘概率密度的乘积
  • 解释：什么样的参数与数据组合后恰好是真实值

对数似然：$\log L(\theta)=\log \prod_{i=1}^{m} \frac{1}{\sqrt{2 \pi} \sigma} \exp \left(-\frac{\left(y^{(i)}-\theta^{T} x^{(i)}\right)^{2}}{2 \sigma^{2}}\right)$(对数中乘法可以转换为加法)

展开化简：

$\begin{array}{l}
\sum_{i=1}^{m} \log \frac{1}{\sqrt{2 \pi} \sigma} \exp \left(-\frac{\left(y^{(i)}-\theta^{T} x^{(i)}\right)^{2}}{2 \sigma^{2}}\right) \
=m \log \frac{1}{\sqrt{2 \pi} \sigma}-\frac{1}{\sigma^{2}} \cdot \frac{1}{2} \sum_{i=1}^{m}\left(y^{(i)}-\theta^{T} x^{(i)}\right)^{2}
\end{array}$

目标：让似然函数越大越好，即下式越小越好：

$J(\theta)=\frac{1}{2} \sum_{i=1}^{m}\left(y^{(i)}-\theta^{T} x^{(i)}\right)^{2}$(最小二乘法)

  • 目标函数：$J(\theta)=\frac{1}{2} \sum_{i=1}^{m}\left(h_{\theta}\left(x^{(i)}\right)-y^{(i)}\right)^{2}=\frac{1}{2}(X \theta-y)^{T}(X \theta-y)$

  • 求偏导：

    $\begin{array}{l}
    \nabla_{\theta} J(\theta)=\nabla_{\theta}\left(\frac{1}{2}(X \theta-y)^{T}(X \theta-y)\right)=\nabla_{\theta}\left(\frac{1}{2}\left(\theta^{T} X^{T}-y^{T}\right)(X \theta-y)\right) \
    =\nabla_{\theta}\left(\frac{1}{2}\left(\theta^{T} X^{T} X \theta-\theta^{T} X^{T} y-y^{T} X \theta+y^{T} y\right)\right) \
    =\frac{1}{2}\left(2 X^{T} X \theta-X^{T} y-\left(y^{T} X\right)^{T}\right)=X^{T} X \theta-X^{T} y
    \end{array}$

  • 偏导为0：$\theta=\left(X^{T} X\right)^{-1} X^{T} y$

注：$X^{T}X$为对称阵。

1.4 梯度下降

机器学习的套路就是我交给机器一堆数据，然后告诉它什么样的学习方式是对的（目标函数），然后让它朝着这个方向去做。

二、KNN算法

KNN(k-nearest neighbor)，即K近邻算法，当需要表示一个样本(值)的时候，就使用与该样本最接近的K个邻居来决定。“近朱者赤近墨者黑”。

算法过程为：

  • 从训练集中选择离待预测样本最近的k个样本
  • 根据这k个样本计算待预测样本的值(属于哪个类别或者一个具体数值)

三、Python数学建模

3.1 线性规划模型

线性规划的本质：

  • 问题是线性的
  • 约束是线性的

线性代数引入向量的目的：不是为了更好的画图，而是为了通过代数的方法解决几何的问题。

数的集合形态是向量，向量的集合形态是矩阵。

3.1.1 矩阵的运算

用Python进行矩阵的运算：

import numpy as np
a = np.array([[1,2,3],[4,5,6]])
b = np.array([[1,2],[3,4],[5,6]])
c = np.array([[1,2,3]])
d = np.array([[9,8,7],[3,2,1]])
#矩阵加法
sum = a + b
#放缩
e = 3 * a
#数乘、矩阵乘
e = np.dot(a,b)
#元素乘
e = a * d
#转置
e = c.T

#逆矩阵
result = np.linalg.inv(e)
#行列式
result = np.linalg.det(e)
#矩阵的秩
e = np.linalg.matrix_rank(d)

3.1.2 一次方程组的解

$\left{\begin{aligned}
10 x-y-2 z &=72 \
-x+10 y-2 z &=83 \
-x-y+5 z &=42
\end{aligned}\right.$

numpy库主要求解数值解，sympy库主要求解符号解。

import numpy as np
A = np.array([[10,-1,-2],[-1,10,-2],[-1,-1,5]])			#A为系数矩阵
b =np.array([72,83,42])								    #b为常数列
inv_A = np.linalg.inv(A)								#A的逆矩阵
x = inv_A.dot(b)										#A的逆矩阵与b做点积运算
x =np.linalg.solve(A, b)								# 5,6两行也可以用本行替代
print(x)
from sympy import symbols, Eq,solve
x,y，z = symbols('x y z')
eqs =[Eq(10 * x-y - 2*z，72),
	  Eq(-x+10*y -2* z,83),Eq(-x -y +5* z,42)]
print(solve(eqs,[x, y, z]))

$ \max z=c^{T} X \ s.t. \left{\begin{array}{l}A \tilde{X}=b \ \tilde{X} \geqslant 0\end{array}\right. $

3.1.4 线性规划的求解策略

线性代数中，解方程组的时候可以用向量和基分解的思想解决的。

  • 单纯形法：固定变量，不断变换基向量求方程组的解带入，观察是否是最优解。

  • 蒙特卡洛法：在可行域范围内生成大批量的随机数据点，观测这些数据点在什么位置取得近似最优。

在线性规划中引入松弛变量：因为单纯形法要求约束条件都为等式（且所有变量非负），不可避免的存在不等式约束。

$\begin{array}{rrrr}
\text { maximize: } & z=x_{1}+2 x_{2} & \text { maximize: } & z=x_{1}+2 x_{2} \
\text { subject to: } & -3 x_{1}+2 x_{2} \leq 3 & \text { subject to: } & x_{3}=3+3 x_{1}-2 x_{2} \
& x_{1}+x_{2} \leq 2 & & x_{4}=2-x_{1}-x_{2} \
& x_{1}-x_{2} \leq 1 & & x_{5}=1-x_{1}+x_{2} \
& x_{1}, x_{2} \geq 0 & & x_{1}, x_{2}, x_{3}, x_{4}, x_{5} \geq 0
\end{array}$

在绝对值问题中，引入$\left| X_{1} - X_{2} \right| \leq 2$，可转为：$X_{1} = X_{2} + \epsilon$，其中$ -2 \leq \epsilon \leq 2$。

3.1.5 二次规划的基本形式

多元函数的极值求解：

每个偏微分都能为0的点就可能是极值点，即：

$\left{\begin{array}{l}
\frac{\partial f}{\partial x}=0 \
\frac{\partial f}{\partial y}=0
\end{array}\right.$

在数学最优问题中，拉格朗日乘子法（Lagrange Multiplier，以数学家拉格朗日命名）是一种寻找变量受一个或多个条件限制的多元函数的极值的方法。

这种方法将一个有n个变量与k个约束条件的最优化问题转换为一个有n + k个变量的方程组的极值问题，其变量不受任何约束。

这种方法引入了一种新的标量未知数，即拉格朗日乘数：约束方程的梯度（gradient）的线性组合里每个向量的系数。

$maximizef (x, y)\ s.t. \quad g(x, y)=0,\mathcal{L}(x, y, \lambda)=f(x, y)-\lambda \cdot g(x, y)$

设目标函数f(x)，不等式约束为g(x)，有的教程还会添加上等式约束条件h(x)。此时的约束优化问题描述如下：

$\begin{array}{c}
\min f(x) \
\text { s.t. }\left{\begin{array}{r}
h(x)=0 \
g(x) \leqslant 0
\end{array}\right.
\end{array}$

$\begin{array}{l}
L(x, \lambda, \mu)=f(x)+\lambda h(x)+\mu g(x) \
\qquad\left{\begin{array}{l}
\left.\frac{\partial L}{\partial X}\right|_{X=X^{}}=0 \
\lambda \neq 0 \
\mu \geqslant 0 \
\mu g\left(X^{}\right)=0 \
h\left(X^{}\right)=0 \
g\left(X^{}\right) \leqslant 0
\end{array}\right.
\end{array}$

根据条件有：$F = F_{G_{1}}+F_{G_{2}}+F_{G_{3}}$，三个不等式关系和$P_{1}+P_{2}+P_{3} = 700$，求最小的$F$。

使用遗传算法：

from sko.GA import GA

def func(x):
    return 10.5+0.3*x[0]+0.32*x[1]+0.32*x[2]+0.0007*x[0]**2+0.0004*x[1]**2+0.00045*x[2]**2

cons = lambda x : x[0]+x[1]+x[2]-700

b1, b2, b3 = (100, 200), (120, 250), (150, 300)
# max_iter为迭代次数
ga = GA(func=func, n_dim=3, size_pop=500, max_iter=500,
        constraint_eq=[cons], lb=[100, 120, 150], ub=[200, 250, 300])
best_x, best_y = ga.run()
print("best x:\n", best_x, "\nbest_y:\n", best_y)

[例]流量的优化问题：

某公司有6个建筑工地要开工,每个工地的位置(用平面坐标系a,b表示,距离单位:千米)及水泥日用量d(吨)由下表给出。规划设立两个料场位于A,B,日储量各为20吨。假设从料场到工地之间均有 直线道路相连，试确定料场的位置,并制定每天的供应计划,即从A,B两料场分别向各工地运送多少吨水泥,使总的吨千米数最小。

$\begin{aligned}
\min f=\sum_{i=1}^{2} \sum_{j=1}^{6} w_{i j} \sqrt{\left(x_{i}-a_{j}\right)^{2}+\left(y_{i}-b_{j}\right)^{2}} \
\text { s.t. }\left{\begin{array}{l}
\sum_{i=1}^{2} x_{i j} \geqslant d_{j} \
\sum_{j=1}^{6} x_{i j} \leqslant e_{i}
\end{array}\right.
\end{aligned}$
