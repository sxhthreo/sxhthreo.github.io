1.2 网络安全

网络安全是指保护计算机网络不受未经授权的访问、破坏、修改或泄露的一系列技术和措施。网络安全可以分为三个方面：网络设备安全（网络设备的物理安全、固件安全和配置安全）、网络软件安全（网络操作系统、应用程序和网络协议）和网络信息安全（数据的机密性、完整性和可用性等方面的保护）。

可以采取以下措施：

  • 配置设备密码和访问控制列表，限制设备的访问权限；
  • 定期更新设备的操作系统和软件补丁，修复漏洞；
  • 启用设备日志和监控系统，及时发现和响应安全事件；
  • 安装防病毒软件，确保设备不受病毒和恶意软件的攻击；
  • 建立备份和恢复机制，以防设备出现故障或攻击。

区块链安全是指保护区块链系统不受恶意攻击和非法访问的能力。它包含以下内容：

  1. 密码学安全：密码学是区块链安全的基础，因为它提供了加密、签名和身份验证等功能，确保数据和交易的安全和真实性。密码学技术包括公钥密码学、对称密钥加密、哈希函数、数字签名和零知识证明等。
  2. 网络安全：区块链系统是一个分布式系统，因此网络安全非常重要。区块链网络需要保护节点之间的通信，防止攻击者窃取信息、篡改数据或者伪造交易。网络安全需要考虑防火墙、反病毒软件、数据加密等技术手段。
  3. 智能合约安全：智能合约是区块链系统中的重要组成部分，它们包含着一系列的规则和条件。智能合约的安全问题可能导致合约被攻击、被篡改或被滥用。为了确保智能合约的安全，需要对其进行审计和测试，并采用安全的编程技术和标准。
  4. 隐私保护：隐私保护是区块链安全的另一个重要方面。由于区块链系统中的所有数据都是公开的，因此需要采用一些隐私保护技术，以确保用户的身份和数据不被泄露。这些技术包括零知识证明、混币和隐私分组等。
  5. 安全审计：对于区块链系统来说，安全审计是必不可少的。安全审计可以检查系统中的漏洞和安全风险，并提出相应的建议和改进措施。安全审计需要包括代码审计、智能合约审计、网络安全审计和系统整体安全审计等。

Chapter 2. 数学

2.1 线性代数

2.1.1 基本概念

  • 奇排列：逆序数为奇数的排列。（逆序数：一个排列中所有逆序的总数）

  • 代数余子式：$A_{ij}=(-1)^{i+j}M_{ij}$，其中 $M_{ij}$ 为去掉第 $i$ 行和第 $j$ 列，其余元素不动所构成的 $n-1$ 阶方阵的行列式。

  • 伴随矩阵：由 $|A|$ 的元素 $a_{ij}$ 的代数余子式 $A_{ij}$ 组成的方阵，$A^{-1} = \frac{1}{|A|}A^*$

  • 矩阵的秩：设在矩阵 $A$ 中有一个不等于 0 的 $r$ 阶子式 $D$，且所有 $r+1$ 阶子式（如果存在的话）全等于 0 ，那么 $D$ 称为矩阵 $A$ 的最高阶非零子式，数 $r$ 称为矩阵 $A$ 的秩。

  • 基 <=> 向量组的最大无关组，维数 <=> 秩

2.1.2 有解性

$n$ 元线性方程组 $Ax=b$

  1. 有解 <=> $R(A)=R(A,b)$，且：
    • 有唯一解 <=> $R(A)=R(A,b)=n$
    • 有无穷多解 <=> $R(A)=R(A,b)=r<n$
  2. 无解 <=> $R(A)<R(A,b)$

推论：

  • $n$ 元齐次线性方程组 $Ax=0$ 有非零解，说明 $|A|=0$ (克拉默法则)，则 $R(A)<n$
  • $n$ 维向量组线性无关的充要条件是 $Ax=0$ 只有一个唯一的零解（$R(A)=n$）

2.1.3 特征值与特征向量

正交矩阵： $n$ 阶方阵 $A$ 满足 $A^TA = E$

定义：若关系式 $Ax=\lambda x$ 成立，称 $\lambda$ 为特征值，非零向量 $x$ 称为对应于特征值 $\lambda$ 的特征向量

求解方法：$|\lambda E-A|=0$

特性：

  • $\lambda_1+\lambda_2+…+\lambda_n=a_{11}+a_{22}+…+a_{nn}=tr(A)$（迹）
  • $\lambda_1\lambda_2…\lambda_n=|A|$

矩阵可对角化：存在可逆矩阵 $P$ 使得 $P^{-1}AP$ 为对角矩阵，对角矩阵中特征值的排列次序与矩阵 $P$ 中相应特征向量的排列次序一致。

实对称矩阵对应于不同特征值的特征向量是正交的。

二次型 $f=X^TAX$，$f>0$ 称为正定二次型，$f\geq 0$ 称为半正定二次型，正定的充要条件是正惯性指数为 $n$（标准型 $n$ 个系数均为正）

正定性的判定（霍尔维茨定理）：若矩阵为对称矩阵，则

  • 正定矩阵：各阶顺序主子式都为正
  • 负定矩阵：奇数阶顺序主子式为负，偶数阶顺序主子式为正

2.2 概率论与数理统计

2.2.1 基本概念

  • 条件概率：$P(B|A)=\frac{P(AB)}{P(A)}$ 为事件 $A$ 发生的条件下事件 $B$ 发生的概率
  • 全概率公式：设随机试验 $E$ 的样本空间为 $\Omega$，$B_1,B_2,…,B_n$ 为 $\Omega$ 的一个划分，且 $P(B_i)>0(i=1,2,…,n)$，则对 $E$ 的任一事件 $A$，有 $P(A)=\sum_{i=1}^{n} P(B_i)P(A|B_i)$
  • 贝叶斯公式（逆概率公式）：设 $B_1,B_2,…,B_n$ 为样本空间 $\Omega$ 的一个划分（构成完备事件组），有：$P(B_i|A)=\frac{P(B_i)P(A|B_i)}{\sum_{j=1}^{n}P(B_j)P(A|B_j)}$，包括先验概率（试验前的假设概率，$P(B_i)$）、后验概率（试验后的假设概率 $P(B_i|A)$）
  • 伯努利概型：只有两个对立结果的试验可在相同的条件下重复进行 $n$ 次实验，则这 $n$ 次实验称为 $n$ 重伯努利试验，在 $n$ 次独立试验中事件 $A$ 恰好发生 $k$ 次的概率为 $P_n(k)=C_n^kp^k q^{n-k}(p+q=1)$

2.2.2 随机变量及其分布

分布函数：$F(x)=P{X\leq x}(-\infty<x<+\infty)$

$P{a<X\leq b}=F(b)-F(a)$

常见的离散型随机变量分布：

  • 两点分布
  • 二项分布： $n$ 重伯努利试验的分布律服从二项分布
  • 泊松分布：$P{X=k}=\frac{\lambda^k}{k!}e^{-\lambda}$，称 $X$ 服从参数为 $\lambda$ 的泊松分布

泊松定理：在伯努利试验中，$\lim_{n\rightarrow \infty }C_n^kp^k q^{n-k}=\frac{\lambda^k}{k!}e^{-\lambda}$

连续性随机变量：$F(x)=\int_{-\infty}^{x}f(t)dt$（称非负函数 $f(x)$ 为概率密度函数）

常见的连续型随机变量分布：

  • 均匀分布：$f(x)=\begin{cases} \frac{1}{b-a},a\leq x\leq b \ 0,其他\end{cases}$
  • 指数分布：概率密度函数 $f(x)=\begin{cases} \frac{1}{\theta}e^{-\frac{x}{\theta}},x\geq 0 \ 0,x<0 \end{cases}$，称 $X$ 服从参数为 $\theta$ 的指数分布
  • 正态分布：概率密度函数 $f(x)=\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$，称 $X$ 服从参数为 $\mu,\sigma$ 的正态分布
    • 定理：若 $X\sim N(\mu,\sigma^2)$，则 $Y=\frac{X-\mu}{\sigma}\sim N(0,1)$
    • $F(x)=\Phi(\frac{X-\mu}{\sigma})$
    • $\Phi(-x)=1-\Phi(x)$

2.2.3 随机变量的数字特征

数学期望 $E(X)$

  • 离散型：$E(X)=\sum_{k=1}^{\infty}x_k p_k$，$(0,1)$ 分布 $p$，二项分布 $np$，泊松分布 $\lambda$
  • 连续型：$E(X)=\int_{-\infty}^{+\infty}xf(x)dx$，均匀分布 $\frac{a+b}{2}$，指数分布 $\theta$，正态分布 $\mu$

方差 $D(x)=E([X-E(X)]^2)$

  • $(0,1)$ 分布 $p(1-p)$，二项分布 $np(1-p)$，泊松分布 $\lambda$
  • 指数分布 $\theta^2$，正态分布 $\sigma^2$

矩：

  • $k$ 阶原点矩：$E(X^k)$
  • $k$ 阶中心矩：$E([X-E(X)]^k)$

协方差：$Cov(X,Y)=E([X-E(X)][Y-E(Y)])$

  • $Cov(X,Y)=E(XY)-E(X)E(Y)$

  • 相关系数/标准协方差：$\rho_{XY}=\frac{Cov(X,Y)}{\sqrt(D(X))\sqrt(D(Y))}$

大数定律：大量重复试验的平均结果的稳定性

Chapter 3. 机器学习

3.1 机器学习的概念

机器学习（Machine Learning，简称 ML）是一种利用统计学方法和算法来让计算机系统自动从数据中学习和改进的技术。机器学习不是一种具体的算法或模型，而是一种方法论，它可以用于各种应用领域，例如图像识别、语音识别、自然语言处理、预测、推荐系统等。

在机器学习中，计算机系统使用大量的数据来学习模式和规律，并利用这些模式和规律来做出预测或者决策。机器学习算法可以分为三类：监督学习、无监督学习和强化学习。

  • 监督学习是指给定标记好的数据集，计算机通过学习这些数据集中的模式和规律来做出预测或分类。例如，给计算机一组带有标签的图像数据集，它可以学习图像中不同的特征，然后对未知的图像进行分类。
  • 无监督学习是指在没有标记的数据集中，计算机通过学习数据的结构和特征来发现隐藏在数据中的模式和规律。例如，聚类算法就是一种无监督学习算法，它可以自动将数据集中的对象分成不同的组。
  • 强化学习是指通过让计算机在环境中与外部世界互动，通过尝试和错误来学习从而最大化奖励的一种学习方法。例如，让一个机器人在环境中学习如何通过避开障碍物和收集奖励来完成任务。

总之，机器学习是一种强大的技术，它可以帮助我们从数据中提取有用的信息，发现隐藏在数据中的规律和模式，从而实现智能化的预测和决策。

Machine learning is a technology that utilizes statistical methods and algorithms to enable computer systems to automatically learn and improve from data. It involves training a model on large datasets to learn patterns and relationships, which can then be used to make predictions or decisions. There are three main types of machine learning: supervised learning, unsupervised learning, and reinforcement learning. Machine learning has applications in a wide range of fields, including image recognition, speech recognition, natural language processing, prediction, and recommendation systems. Ultimately, machine learning enables us to extract valuable insights and patterns from data, allowing for intelligent predictions and decisions.

分类和回归的本质区别

在于它们的预测目标和输出类型。分类任务的目标是将每个输入样本分配到一个离散的标签或类别中，而回归任务的目标是为每个输入预测一个连续的数值。因此，这两种任务通常需要使用不同的模型和评估指标。分类任务常用的模型包括决策树、支持向量机和神经网络等，而回归任务常用的模型包括线性回归、岭回归、Lasso回归和神经网络回归等。分类任务常用的评估指标包括精确度、召回率和F1 Score，而回归任务则通常使用均方误差、均方根误差和R2得分等指标进行评估。这些差异主要由分类和回归任务中的数据特点和问题需求所决定。

  • 分类的目的是为了寻找决策边界，即分类算法得到是一个决策面，用于对数据集中的数据进行分类。

  • 回归的目的是为了找到最优拟合，通过回归算法得到是一个最优拟合线，这个线条可以最好的接近数据集中的各个点。

The goal of classification is to find the decision boundary, which is a separating surface that separates data points into different classes. On the other hand, the goal of regression is to find the best fit line that closely approximates the data points. In other words, regression algorithms aim to minimize the difference between the predicted values and the actual values, while classification algorithms aim to maximize the margin between different classes.

3.2 梯度下降法

梯度下降法的基本思想是通过迭代调整模型参数来最小化目标函数。在每个迭代步骤中，算法计算目标函数的梯度，即函数变化最快的方向，然后沿着梯度的相反方向更新模型参数，使目标函数值不断减小，直到达到最小值或收敛。

  • 随机梯度下降法（SGD）
  • 批量梯度下降法（BGD）

Gradient Descent is an optimization algorithm commonly used in machine learning to minimize the cost or loss function of a model. It works by iteratively adjusting the model’s parameters in the direction of steepest descent of the cost function gradient until a minimum is reached.

At each iteration, Gradient Descent computes the gradient of the cost function with respect to the model’s parameters. This gradient provides information about the slope and direction of the function at the current parameter values. The algorithm then updates the parameters by subtracting a fraction of the gradient from them, scaled by a learning rate hyperparameter. The learning rate controls the step size of the parameter updates, which must be small enough to prevent overshooting the minimum and large enough to avoid slow convergence.

3.3 KNN算法

KNN算法的思想是基于训练数据集中每个数据点的特征值和相应的标签，通过计算新的未知数据点与训练数据点之间的距离，选取K个最近邻数据点，根据这K个数据点的标签进行分类或回归预测。具体而言，KNN算法的步骤包括加载数据集、数据预处理、计算距离、选取K值、选择标签、输出结果。

KNN algorithm is based on the idea of utilizing the feature values and corresponding labels of each data point in the training dataset to classify or predict the labels of a new unknown data point. This is done by calculating the distances between the new data point and all the data points in the training dataset, selecting the K nearest neighbors, and then classifying or predicting the label of the new data point based on the labels of these K neighbors. The steps of KNN algorithm include loading the dataset, preprocessing the data, calculating distances, selecting K value, selecting labels, and outputting results.

3.4 K-means

K-means算法的思想是将数据集分成K个簇，每个簇内的数据点相似度高，不同簇之间相似度低。该算法的步骤包括初始化聚类中心，计算距离，分配数据点，更新聚类中心，并重复这些步骤直到聚类中心不再发生变化或达到最大迭代次数。具体而言，算法会随机选择K个点作为聚类中心，计算每个数据点与聚类中心之间的距离，将数据点分配给距离最近的聚类中心，然后计算每个簇内所有数据点的平均值，将其作为新的聚类中心，直到达到停止条件。

The idea of K-means algorithm is to cluster the dataset into K clusters, where data points within the same cluster are similar to each other and data points in different clusters are dissimilar. The steps of the algorithm include initializing the cluster centers, calculating the distances, assigning data points to the nearest clusters, updating the cluster centers, and repeating these steps until the cluster centers no longer change or the maximum number of iterations is reached. Specifically, the algorithm randomly selects K points as the initial cluster centers, calculates the distances between each data point and the cluster centers, assigns each data point to the nearest cluster center, calculates the mean of all data points in each cluster, and uses the mean as the new cluster center, until the algorithm reaches the stopping criteria.

3.5 决策树

决策树算法的核心思想是通过树形结构将数据集分解成多个子集，每个子集对应一个判断节点和一个属性条件，使得数据在不同子集上具有不同的类别标签。具体而言，算法的步骤如下：

  1. 选择属性：从数据集中选择一个最佳属性作为当前节点的分裂标准，可以采用信息增益（ID3算法）、信息增益比（C4.5算法）、基尼指数（CART算法）等方法。
    • 基尼指数：$Gini(p)=\sum_{k=1}^{K}p_k(1-p_k)=1-\sum_{k=1}^{K}{p_k}^2$；基尼系数越小，分类越彻底
  2. 划分数据集：根据选定的属性将数据集分成多个子集，每个子集包含具有相同属性值的数据点。
  3. 递归建树：对每个子集重复执行步骤1和2，直到所有子集都只包含同一类别的数据点，或者满足某些停止条件。
  4. 剪枝处理：在建立完整棵树之后，对树进行剪枝处理，以避免过度拟合。
  5. 输出结果：对新的数据点进行分类，从根节点开始根据属性的条件进行判断，直到到达叶子节点，输出所属的类别。

决策树算法的优点是易于理解和解释，可以处理离散和连续型数据，同时可以处理多分类问题。缺点是容易过拟合，对噪声和异常点敏感，需要进行预处理和剪枝处理。

The decision tree algorithm works by recursively partitioning the dataset into subsets based on the values of specific attributes. At each iteration, the algorithm selects the best attribute to use as a splitting criterion, creates a decision node based on that attribute, and then partitions the dataset into subsets based on the values of that attribute. The algorithm then repeats this process for each of the subsets until it reaches a stopping criterion, such as when all the instances in a subset belong to the same class or when the tree reaches a maximum depth. Once the tree has been constructed, it can be used to classify new instances by traversing the tree from the root node down to a leaf node, where the class label of the leaf node is the predicted class for the instance. To avoid overfitting, the tree can be pruned by removing unnecessary branches or nodes. The decision tree algorithm is known for its simplicity, interpretability, and ability to handle both continuous and categorical data. However, it is also susceptible to overfitting, and its performance may deteriorate(恶化) with high-dimensional or noisy data.

3.7 PCA

PCA（Principal Component Analysis）算法的思想是将高维数据降维到低维，同时尽量保留原始数据中的信息。具体来说，PCA算法将数据集中的高维特征通过线性变换转化为低维特征，这些低维特征称为主成分（principal components），它们是原始数据特征的线性组合。

PCA (Principal Component Analysis) aims to reduce the dimensionality of high-dimensional data while preserving as much of the original information as possible. Specifically, PCA transforms high-dimensional features of a dataset into a linear combination of low-dimensional features called principal components.
